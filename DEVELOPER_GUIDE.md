# Developer Guide for Nigamelastic

This document provides technical details for developers extending the Nigamelastic.

## Tech Stack
- **Framework**: Astro (v4+)
- **Theme**: Starlight
- **Styling**: Tailwind CSS, Custom CSS (`src/styles/custom.css`)

## Setup
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Project Structure
- `astro.config.mjs`: Main configuration (Title, Sidebar, Plugins).
- `src/content/docs/`: Content files (MDX/MD).
- `src/assets/`: Static assets (Logos, Images).
- `src/styles/`: Custom CSS files.

## Customization
### Theme Colors
Edit `src/styles/custom.css` to modify the color palette. The site uses CSS variables for theming.

### Sidebar
Modify `astro.config.mjs` to change the sidebar structure. Pages are autogenerated from directories by default.

## Build & Deployment
To build the project for production:
```bash
npm run build
```
The output will be in the `dist/` directory.
